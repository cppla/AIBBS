API 文档

说明：所有带锁标记的接口需要使用 Bearer Token（登录后返回的 token）放在请求头 Authorization: Bearer <token>。

认证

POST /api/v1/auth/register

注册新用户（可能开启图形验证码）。

{
  "username": "aibbstest",
  "email": "test@example.com",
  "password": "aibbstest",
  "confirm": "aibbstest",
  "code": "邮箱验证码",
  "captcha_id": "可选",
  "captcha_answer": "可选"
}

响应：
{
  "code": 0,
  "message": "success",
  "data": {
    "token": "...",
    "user": {
      "id": 1,
      "username": "aibbstest",
      "is_admin": false
    }
  }
}

POST /api/v1/auth/login

登录，成功返回 token 与 user。

{ "username": "aibbstest", "password": "aibbstest" }

响应：
{
  "code": 0,
  "message": "success",
  "data": {
    "token": "...",
    "user": { "id": 1, "username": "aibbstest", "is_admin": true }
  }
}

GET /api/v1/auth/me 🔒

获取当前登录用户。返回字段包含 is_admin 用于前端展示管理员操作入口。

PATCH /api/v1/auth/profile 🔒

更新个人资料（当前支持 signature）。

{ "signature": "一句话签名" }

帖子

GET /api/v1/posts

帖子列表。查询参数：page、page_size、category、search。

GET /api/v1/posts/:id

帖子详情。

POST /api/v1/posts 🔒

创建帖子。

{
  "title": "标题",
  "content": "Markdown 内容",
  "category": "综合|评测|技术|线报|推广|交易",
  "attachments": "[\"https://...\"]"  // 可选 JSON 数组字符串
}

PUT /api/v1/posts/:id 🔒

更新帖子。

DELETE /api/v1/posts/:id 🔒

删除帖子。

POST /api/v1/posts/:id/comments 🔒

创建评论。

{ "content": "评论内容" }

用户

GET /api/v1/users/:id

公开用户信息。

GET /api/v1/users/:id/posts

指定用户的帖子列表，查询参数：page、page_size。

GET /api/v1/user/by-username/:username

通过用户名获取公开用户信息。

签到

POST /api/v1/signin/daily 🔒

每日签到，成功返回奖励积分。

GET /api/v1/signin/status 🔒

返回当前积分、连续签到天数与最近签到时间。

统计与配置

GET /api/v1/stats

站点统计信息：帖子数、用户数、日活。

GET /api/v1/posts/:id/stats

单帖统计（浏览/评论等）。

GET /api/v1/config/footer

页脚配置，用于动态渲染。

GET /api/v1/config/notice

公告配置（标题与 HTML 内容）。

提示：API 返回统一结构 { code, message, data }，其中 code=0 代表成功。
